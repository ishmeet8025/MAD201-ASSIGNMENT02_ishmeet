# MAD201-ASSIGNMENT02-ISHMEET

**Lab Title:** Cross-Platform Mobile Portfolio App  
**Student Name / ID:** Ishmeet Singh / A00202436  

## Project Description
This project is a **React Native & Expo mobile portfolio app** that showcases my profile, skills, projects, and contact information.  
It demonstrates multi-screen navigation, reusable components, and Flexbox-based layout styling.  

## Screens
- **Home Screen:** Displays profile picture, name, short bio, and navigation buttons to Projects and Contact screens.  
- **Projects Screen:** Displays a list of sample projects using reusable `ProjectCard` components.  
- **Contact Screen:** Displays social links (LinkedIn, GitHub) and email contact.  

## Tech Stack
- React Native (with Expo)  
- TypeScript (TSX)  
- Expo Router / React Navigation  
- Flexbox & StyleSheet for layouts
